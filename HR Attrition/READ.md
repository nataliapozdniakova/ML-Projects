# HR Employee Attrition Prediction

## Project Overview
This project analyzes HR employee data to predict attrition and uncover factors that influence employee turnover. 
The goal is to provide actionable insights for HR and management teams to reduce attrition and improve workforce planning.

---

## Dataset
- **Source:** [IBM HR Analytics Employee Attrition & Performance - Kaggle](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)
- **Description:** Contains anonymized HR data including employee demographics, job roles, performance metrics, and attrition labels.

---

## Tech Stack
- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn  
- **Tools:** Jupyter Notebook, GitHub

---

## Project Workflow
1. **Data Cleaning & Preprocessing**  
   - Handle missing values and incorrect data types  
   - Encode categorical variables  

2. **Exploratory Data Analysis (EDA)**  
   - Visualize distributions and relationships  
   - Identify trends and correlations  
   - Summarize key insights  

3. **Feature Engineering**  
   - Create new variables to capture patterns  
   - Normalize and scale numerical features  

4. **Machine Learning Models**  
   - Train classification models to predict attrition  
   - Evaluate performance using metrics like accuracy, precision, recall, and F1-score  

5. **Insights & Recommendations**  
   - Highlight most important features influencing attrition  
   - Suggest actionable strategies for HR  



## Skills Demonstrated
- Data wrangling and preprocessing  
- Exploratory data analysis (EDA)  
- Feature engineering for predictive modeling  
- Machine learning model training, evaluation, and interpretation  
- Data visualization and storytelling
